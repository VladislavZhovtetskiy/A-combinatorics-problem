# Water Balance Calculator 💧

A simple Python program that helps calculate the daily water intake based on personal body parameters and activity levels.

## Features
* Validates user inputs (height, weight, age, and cup volume) to prevent incorrect data.
* Custom calculations for both male and female users.
* Dynamic water volume calculation depending on the user's daily activity level (low, mid, high).
* Interactive loop to run multiple calculations without restarting the script.

## Technical Stack
* Python 3
* Functions & Data Decomposition
* Exception Handling (`try-except` blocks)
* Data structures (Dictionaries)
