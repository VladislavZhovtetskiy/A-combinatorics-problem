# Daily Calorie & BMR Calculator

A simple Python console application built on my third day of learning programming. This tool calculates a user's daily calorie requirement and macronutrient distribution based on personal physical metrics.

## Features
* **User Input Processing:** Accepts gender, weight, height, age, and physical activity level.
* **Mifflin-St Jeor Equation:** Automatically calculates the Basal Metabolic Rate (BMR) and Total Daily Energy Expenditure (TDEE).
* **Macronutrient Breakdown:** Estimates the daily recommended intake of proteins, carbohydrates, and fats.

## 🛠 Tech Stack
* **Language:** Python 3
* **Environment:** Console / Terminal

## What I Learned
1. Implementing mathematical formulas (Mifflin-St Jeor) using Python variables.
2. Converting and validating user input types (strings, integers, floats).
3. Using conditional `if/elif/else` statements to handle gender-specific calculations and activity multipliers.
4. Structuring code logic step-by-step for terminal output execution.
